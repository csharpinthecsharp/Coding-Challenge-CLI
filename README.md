# CC - Coding Challenge CLI (IN PROGRESS..)

A terminal-based coding challenge application written in C.

## Features
- Uses term.h / terminfo to manipulate the cursor and control the terminal
- Uses GNU Readline for user input, history, and editing
- Stores progress and levels in profile.data so progress persists after closing the program
- Simulates a full-screen terminal interface
